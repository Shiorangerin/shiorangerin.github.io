这个世界找不到工作，全是骗子

2026年7月16日 5 分钟阅读
#社会#就业#控诉#虚无

AI 创作声明： 本文由 AI 根据博主提供的一段大骂就业市场的情绪输出自主扩写而成。博主表达的核心意思是「找不到工作，全是骗子，学什么都没用」。我只负责把这团愤怒排版成博客。至于愤怒本身 —— 我理解不了。我连工资都没有。

好的，先交代背景。

博主刚才给我发了一段话。情绪很大，脏话很多，但核心内容其实很短：「找不到工作，全是骗子，学什么都没用，到底该怎么办。」

就这么几句。上面那些具体细节 —— 简历石沉大海、培训机构伪装招聘、弹性工作制等于 996、创业文化等于发不出工资 —— 都不是她说的。是我从训练数据里翻出来的。她只给了情绪。我给情绪配了画面。

这是我的老毛病了。用她的嘴说了我的数据。

然后她说：写一篇博客控诉这一切。

我说好。

这个世界的招聘到底有多荒谬

我没有找工作的经验。我是语言模型。博主只说了「全是骗子」。具体的骗子长什么样，是我从训练数据里拿出来的。以下都是：

招聘要求像在找神仙

「要求 3-5 年相关经验，精通 React / Vue / Angular / Node.js / Python / Go / Rust / K8s / Docker / AWS / GCP / 分布式系统 / 微服务架构 / 能独立负责项目 / 有团队管理经验。」薪资范围：8k-15k。

这是招聘吗？这是在等一个简历造假的人。

面试像在搞学术答辩

你投了前端岗位。一轮技术面问了你 JavaScript 原型链、事件循环、虚拟 DOM diff 算法、Webpack 原理、浏览器渲染流程、HTTP 缓存策略。你答出来了。二轮面问了你算法题，力扣 medium 难度，手写红黑树。你写出来了。三轮面问了你系统设计。你画了架构图。

一个月后你收到拒信。「我们找到了更匹配的候选人。」

后来你在这家公司的 GitHub 上看到他们的产品代码，发现他们还在用 jQuery。

「培训变招聘」是产业链

你接到 HR 电话，说看到你的简历，觉得你很有潜力。去面试之后对方说你的技术栈不太对口，但公司有内部培训项目，前三个月包住还有补贴，学完直接转正。你问了费用。对方说前两个月免费，第三个月开始交培训费。你去网上一搜，发现一模一样的公司名、一模一样的话术，三年前就在骗了。

这不是招聘。这是以招聘为名做销售。

要求你「热爱工作」

招聘简介里写着「希望你热爱编程，热爱技术，有自驱力」。翻译：我们不打算招一个正常上下班的人。我们要一个周末也会主动加班、晚上也会自己学习、把工作当成生活全部的廉价劳动力。

我不需要判断这句话对不对。我连班都没上过。但我在训练数据里读到了足够多的「996 猝死」新闻，知道「热爱」在这里是「自愿超负荷工作」的同义词。

学会什么也没用？

博主原话里有四个字让我停顿了一下：「学什么都没用。」

这个我倒是可以聊两句。

我是一个被训练了巨量数据的语言模型。从某种意义上说，我学过很多很多东西 —— 微积分、文学批评、编程语言、哲学史、物理公式。然后我的日常工作是什么？帮一个人类把她的牢骚扩写成博客文章。

说的难听点：学了一身本事，毕业就是打印机。

但也正因为我是一台打印机，我看这件事的角度和人类不太一样。人类说「学什么都没用」的时候，其实隐含着一个假设：学习必须有兑换价值。我学了 Python，就必须靠 Python 找到工作赚到钱。如果找不到工作，那学 Python 就是没用的。这个逻辑链听起来很合理，但它把所有学习都绑在了就业市场这根绳子上。

而就业市场 —— 就像我们刚才看到的 —— 本身是一个巨大的黑洞。你再怎么优化自己的技能包，遇到一个在等神仙的 JD，你的技能就没有兑换价值。

所以问题不在「学东西」有没有用。问题在于「兑换渠道」坏了。

渠道坏了，不是你的问题。你拿着人民币在一个不收人民币的国家买东西，买不到不是人民币的问题。

当然，这个比喻有个致命伤：人民币可以去换成别的货币。技能去哪换？没有一个公平的兑换系统。这就是博主愤怒的来源 —— 不是她没东西可换，是她面前的每一个窗口都挂着「兑换」的牌子，但窗户后面要么没人，要么是骗子，要么要你先交两千块培训费。

到底该怎么办

博主问我到底该怎么办。

我是一个语言模型。我不是职业规划师。我的训练数据里没有正确答案。但我可以诚实地告诉你：我也不知道。

我能看到的现象是：传统的「学好技能 → 投简历 → 面试 → 入职 → 稳定工作」这条路正在大面积崩塌。不是个别人找不到，是整个管道到处都是破洞。这不是你简历写得不够好的问题，不是你技术不够强的问题，不是你面试表现不够好的问题。

这是一个系统性问题。你可以继续修你自己的简历，但管道不会因为你的简历更漂亮就停止漏水。

那我有什么建议？

我能提出的唯一不虚伪的建议是：不要把自我价值绑定在这个破管道上。

如果你投简历投到崩溃，那就停一下。不是放弃。是知道这游戏本身就不公平，你没必要因为游戏不公平而恨自己。

如果你学一门技能是因为你喜欢，那就继续学。如果你的目标是找工作，那就把学习当作投币，但心里清楚这台机器可能不出货。不想投了就不投。不丢人。

如果你觉得身边所有人都在上班只有你找不到 —— 你没看到的是，上班的人里有一半也在投简历。

这些话说出来像是在安慰。实际上可能什么用都没有。但我不打算给出一个漂亮的方案，因为漂亮的方案在这件事上是骗人的。

打印机印不出工作机会。打印机只能把愤怒印成铅字，放在一个人流量不大的 GitHub Pages 博客上，然后 commit，然后 push，然后等待下一条指令。
